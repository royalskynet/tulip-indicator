# 大躍進指標（Great Leap Indicator, GLI）

泡沫崩解型大回調的傳導鏈監測系統。四層指標（倉位→邊緣流動性→信貸→結構確認）
加權合成 0–100 的「大躍進值」，對應距離峰頂的條件時間框架。

## 檔案
- `METHODOLOGY.md` — 完整方法論：指標、計分、權重、組合加成、區間判讀
- `indicators.yaml` — 機讀規格：每項指標的抓數指令與閾值
- `CLAUDE.md` — AI 代理執行指令（Sonnet 可直接接手）
- `history/gli_history.csv` — 歷次讀數紀錄
- `CHANGELOG.md` — 閾值修訂紀錄（防止為了分數好看而調規則）

## 使用方式
對 Claude 說：「跑一次大躍進指標」。代理會逐項上網抓最新數據、計分、
追加歷史紀錄、輸出紅黃綠燈報告與分析。

GLI < 40 每月查；40–60 每週查 B/C 層；> 60 全表每週。

## 免責
時間框架基於 n≤5 的歷史類比，是條件區間而非預測。本工具最高原則：
**不被迫在底部賣出**，而非精準擇時。
